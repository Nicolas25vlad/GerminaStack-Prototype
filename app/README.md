# GerminaStack Vanilla

Site estatico em HTML, CSS e JavaScript vanilla.

## Como abrir

Abra `index.html` diretamente no navegador ou rode:

```bash
npm run dev
```

Depois acesse `http://localhost:5173`.

## Validacao

```bash
npm run check
```

## Estrutura

- `index.html`: documento principal
- `styles.css`: design system e responsividade
- `script.js`: dados, renderizacao e interacoes

## Telas

- Feed inicial
- Explorar
- Notificacoes
- Salvos

O projeto nao usa React, Vite, Tailwind, CDNs, imagens remotas ou bibliotecas externas.
